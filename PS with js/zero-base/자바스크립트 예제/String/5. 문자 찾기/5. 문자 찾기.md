# 문자 찾기

한 개의 문자열을 입력받고, 특정 문자를 입력받아 해당 특정문자가 입력받은 문자열에 몇 개 존재하는지 알아내는 프로그램을 작성하세요. 문자열의 길이는 100을 넘지 않습니다.

### 사고

1. new RegExp 생성자 함수를 사용하여 정규식 패턴을 사용자 입력과 같은 다른 원본에서 가져옵니다.
