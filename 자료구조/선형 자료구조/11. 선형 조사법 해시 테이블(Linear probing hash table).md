# 선형 조사법 선형 조사법 해시 테이블(Linear probing hash table)

- Hash 충돌이 발생했을 때, **그 다음 주소를 확인하고 비어 있다면 그 자리에 대신 저장**하는 해시테이블 기반 자료구조

![](./Linear_probing_hash_table/images/LinearProbing.png)
