# 스택(Stack)
나중에 넣은 데이터가 먼저 나오는 **LIFO(Last In First Out)** 기반의 선형 자료 구조

- 배열과 함께 자료구조에서 많이 쓰이는 자료구조
- 예시) `ctrl + z` 기능, 브라우저 뒤로가기 기능
